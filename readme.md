# EisenhauerKanban

## Run on Mac:

open Terminal in root directory

### Backend:

- run: cd Backend
- run: npm install
- run: nodemon server.js

run: cd ..

### Frontend:

- install xcode
- install iOS 17.5 in xcode
- run simulator with iOS 17.5
- run: cd Frontend
- run: npm install
- run: react-native run-ios
